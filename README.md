# aaa's user-base v1

# NPM Packages required
- ├── discord-interactions@^3.4.0
- └── discord.js@^14.14.1

# How to setup
## Do this:
- Add your bot's ID inside of `config.json`
## (Self)hosted:
- Add your bot's token inside of `config.json`
## For replit:
- Make a secret named `token` and put your token inside instead.
